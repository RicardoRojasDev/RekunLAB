# 🧪 Rekün LAB — Ecommerce 3D Sustentable

Ecommerce desarrollado para Rekün LAB, empresa chilena enfocada en reciclaje de plástico, fabricación de filamento PLA ecológico, impresión 3D y venta de tecnología.

---

## 🚀 Stack Tecnológico

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Supabase (DB, Auth, Storage)
- PostgreSQL

---

## ♻️ Enfoque del Proyecto

- Economía circular real
- Producción local en Chile
- Tecnología + sustentabilidad
- Ecommerce escalable (B2C + B2B)

---

## 🛒 Funcionalidades

### Cliente
- Catálogo dinámico
- Detalle de producto
- Carrito de compras
- Checkout
- Compra como invitado
- Login con Google
- Pedidos
- Correos transaccionales
- Cotizaciones

### Admin
- Panel administrativo
- CRUD productos
- Gestión pedidos
- Gestión imágenes
- Gestión catálogo

---

## 🧩 Tipos de Productos

- Filamentos PLA
- Impresoras 3D
- Packs
- Productos personalizados

---

## 🧱 Arquitectura

- Modular
- Escalable
- API-first
- Separación de responsabilidades
- Sin lógica duplicada

---

## 💰 Reglas de Negocio

- Precios con IVA incluido
- Solo ventas en Chile
- Solo envíos
- Stock no visible
- Cotizaciones vía formulario

---

## 🔐 Backend

- Supabase como fuente de verdad
- PostgreSQL
- RLS para seguridad
- Storage para imágenes
- Auth para usuarios
